Gestión de Eventos

Prueba de conocimientos, presentada para desarrollador de software de Finaktiva

Sobre la prueba

El objetivo de esta prueba técnica es implementar una aplicación de gestión de eventos “Logs”.

La aplicación permite a los usuarios realizar las siguientes acciones:

  1. Registrar un evento en la base de datos mediante un endpoint con ApiGateway.
  2. Un formulario de registro de eventos manuales.
  3. Consultar la información de los eventos registrados en la base de datos
  4. El formulario de consulta debe tener filtros por tipo de evento “Api o Formulario de eventos manuales” y rango de fechas
